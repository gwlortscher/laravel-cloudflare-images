## Installation

Install via composer
```bash
composer require koraycicekciogullari/laravel-cloudflare-images
```

## Usage

Add To .env File

```bash
CLOUDFLARE_IMAGES_ACCOUNT=''
CLOUDFLARE_IMAGES_KEY=''
```
